jQuery Wikipedia Plugin

jQuery Plugin zum einfachen nachladen von Daten aus Wikipedia über die Wikipedia API mittels JSON.
Zum laden muss ein Wikipediaseitentitel übergeben werden. Über den zweiten Parameter können
diverse Einstellungen für die Darstellung getroffen werden.
by Marcel Grolms - www.suabo.de 2013

Tipp: Wenn Sie die Bilder nicht sehen können, liegt es daran, das Sie das Beispiel lokal geladen haben und die Bildpfade von Wikipedia für die SSL verschlüsslung unabhängig mit "//" gesetzt wurden und daher nicht gefunden werden können.

Nutzung des Plugins

      
$('#ContainerToLoadWikipediaContent').WikipediaWidget('Page Title', [{ 'showTitle': true, 
                                                                       'maxThumbnails': 3, 
                                                                       'cutFirstInfoTableRows': 4, 
                                                                       'maxInfoTableRows': 8, 
                                                                       'thumbMaxWidth': '180px', 
                                                                       'thumbMaxHeight': '180px' 
                                                                     }]);

einfacher Aufruf ohne Optionen

Beispielcode: 
    $('#demo1').WikipediaWidget('Stuttgart');

Begrenzung der Thumbnails auf 6 erhöhen und keinen Titel anzeigen

Beispielcode: 
    $('#demo2').WikipediaWidget('Albert Einstein', { 'showTitle' : false, 'maxThumbnails' : 6 });
